# Installation
The recommended way to install blink is using [Rokit](https://github.com/rojo-rbx/rokit).
<Steps>

### Installing Blink

Open a terminal in your project directory and run the following command:

```sh copy filename="Bash"
rokit add 1Axen/blink
```

### Updating Blink

When a new version of blink releases, using Rokit you can easily update using the following command:

```sh copy filename="Bash"
rokit update 1Axen/blink
```

</Steps>

## Alternatives

<details>
<summary>GitHub Releases</summary>

You can download pre-built binaries directly from the
[GitHub Releases](https://github.com/1Axen/blink/releases) page.

</details>

<details>
<summary>pesde (0.18.5+)</summary>

An official release is available on pesde starting from version 0.18.5.  
To install blink using pesde, open a terminal in your project directory and run the following commands:
```sh copy filename="Bash"
pesde add --dev 1axen/blink
pesde install
```

</details>

<details>
<summary>Roblox Studio</summary>
Blink offers a companion studio plugin which allows you to write and generate files within Studio without the need for external tooling.  
The studio plugin can be installed from the [Creator Store](https://create.roblox.com/store/asset/77231976488966/Blink-Editor).  
Alternatively you can download it from [Github Releases](https://github.com/1Axen/Blink/releases).
</details>

<details>
<summary>Bytecode</summary>

Blink releases also contain bundled bytecode which can be used to run blink.
<Steps>

### Downloading Bytecode
You can download the bundled bytecode from [Github Releases](https://github.com/1Axen/Blink/releases).
Once downloaded unzip it into your target directory.

### Installing Lune
Blink uses lune as it's runtime enviornment, you can install lune using Rokit:
```sh copy filename="Bash"
rokit add lune
```

### Running bytecode
Open the directory in which you unzipped the bytecode and run the following command:
```sh copy filename="Bash"
lune run init [INPUT] -- [ARGS]
```

</Steps>
</details>
